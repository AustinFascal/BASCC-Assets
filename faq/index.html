Umum Akun/Profil

Saya tidak memiliki koneksi internet, apakah saya masih bisa menggunakan aplikasi?

Anda masih bisa melihat data secara offline, namun tidak bisa melakukan operasi berupa membuat atau update data apapun, seperti membuat Good Issue, Jurnal Good Issue, Cash Out, Invoice, dan AIO. Untuk saat ini, operasi berupa lihat detail dan hapus data dapat dijalankan secara offline. Adapun operasi hapus data akan di-update di database saat Anda memiliki koneksi internet kembali.
